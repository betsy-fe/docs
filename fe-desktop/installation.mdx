## Prerequisites

Before you begin, ensure you have the following installed:

<AccordionGroup>
  <Accordion title="Node.js v22.17.0 or higher" icon="node-js">
    Use [fnm](https://github.com/Schniz/fnm) or [nvm](https://github.com/nvm-sh/nvm) to manage Node.js versions.
    
    ```bash
    # Using fnm
    fnm install 22.17.0
    fnm use 22.17.0
    
    # Using nvm
    nvm install 22.17.0
    nvm use 22.17.0
    ```
  </Accordion>
  
  <Accordion title="pnpm package manager" icon="box">
    Install pnpm globally:
    
    ```bash
    npm install -g pnpm
    ```
  </Accordion>
</AccordionGroup>

## Global dependencies

Install these packages globally for smooth project operation:

```bash
pnpm i -g npm-run-all rimraf
```

<Note>
  These global packages ensure the project runs smoothly even after `node_modules` reinstallation.
</Note>

## Clone the repository

```bash
git clone https://github.com/betsy-fe/fe-desktop.git
cd fe-desktop
```

## Environment setup

1. Copy the example environment file:

```bash
cp .env.example .env
```

2. Edit `.env` and configure the required variables:

```bash .env
# API Configuration
API_URL=https://api.example.com
SOCKET_URL=wss://socket.example.com

# Client Configuration
CLIENT_ID=your_client_id
DEFAULT_LOCALE=en

# Feature Flags
ENABLE_DEMO_MODE=true
ENABLE_THEME_EDITOR=false
```

<Warning>
  Never commit your `.env` file to version control. It contains sensitive configuration.
</Warning>

## Install dependencies

```bash
pnpm install
```

This will install all project dependencies defined in `package.json`.

## Verify installation

Run the development server to verify everything is set up correctly:

```bash
pnpm start
```

The application should start at [http://localhost:8080](http://localhost:8080).

<Check>
  If you see the betting interface, your installation is successful!
</Check>

## Common issues

<AccordionGroup>
  <Accordion title="Port 8080 already in use" icon="triangle-exclamation">
    If port 8080 is occupied, you can change it in `webpack.config.dev.ts`:
    
    ```typescript
    devServer: {
      port: 3000, // Change to your preferred port
    }
    ```
  </Accordion>
  
  <Accordion title="Node version mismatch" icon="triangle-exclamation">
    Ensure you're using Node.js v22.17.0 or higher:
    
    ```bash
    node --version
    ```
    
    If the version is incorrect, use fnm or nvm to switch versions.
  </Accordion>
  
  <Accordion title="pnpm install fails" icon="triangle-exclamation">
    Try clearing the cache and reinstalling:
    
    ```bash
    pnpm store prune
    rm -rf node_modules pnpm-lock.yaml
    pnpm install
    ```
  </Accordion>
</AccordionGroup>
