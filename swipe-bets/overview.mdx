## Overview

Swipe Bets is a mobile-first betting interface that allows users to quickly place bets by swiping through personalized betting recommendations. The feature provides a Tinder-like card interface for rapid bet placement with preset or maximum bet amounts. The primary value of Swipe Bets is reducing time-to-bet on mobile to a single gesture.

## What it does

Swipe Bets enables users to:

- Browse personalized betting recommendations in a card-based interface
- Place bets by swiping right on cards
- Skip recommendations by swiping left
- Undo the last swipe action
- Set custom bet amounts or use maximum bet calculations
- View real-time odds and potential winnings

## Why it exists

The feature addresses several key user and business needs:

- Faster bet placement in mobile-first scenarios
- Simplified betting flow for casual and less experienced users
- Increased engagement through personalized recommendations
- Reduced interaction cost compared to traditional betslip flows
- Improved overall mobile betting experience
