## Overview

Swipe Bets is a mobile-first betting interface that allows users to quickly place bets by swiping through personalized betting recommendations. The feature provides a Tinder-like card interface for rapid bet placement with preset or maximum bet amounts.

## What it does

Swipe Bets enables users to:
- Browse personalized betting recommendations in a card-based interface
- Place bets by swiping right on cards
- Skip recommendations by swiping left
- Undo the last swipe action
- Set custom bet amounts or use maximum bet calculations
- View real-time odds and potential winnings

## Why it exists

The feature addresses the need for:
- Faster bet placement on mobile devices
- Simplified betting workflow for casual users
- Personalized betting recommendations
- Reduced friction in the betting process
- Improved mobile user experience
